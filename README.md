## The Method: 3-Phase Buildout

Every engagement follows the same disciplined architecture:

### Phase 1 — Forensic Audit *(Diagnosis)*
- **Operational mapping** — Where does time leak? Money leak? Decisions stall?
- **Readiness scoring** — Proprietary 100-point scale across 8 modules (🔴 🟡 🟢)
- **GO / NO-GO Verdict** — One clear answer. No ambiguity.
- **Critical blockers** — The 3-5 things that must be true before any spend.

### Phase 2 — Core Infrastructure *(Buildout)*
- **Nashville Architecture™** — Single source of truth for every process, KPI, and owner.
- **SOP Engine** — Processes that survive the founder leaving the room.
- **KPI Scorecards** — The 5-10 numbers that predict business health.
- **Accountability Rhythm** — Daily sign-offs, weekly reviews, 90-day sprints.

### Phase 3 — Scale & Sustain *(Optimization)*
- Revenue architecture and offer sequencing
- Team scaling by role design, not desperation
- Handoff protocol — the system runs without us

## Key Terminology

| Term | Meaning |
|---|---|
| **Forensic Audit** | Full-spectrum operational diagnosis before any buildout |
| **Nashville Architecture** | Single source of truth framework — processes, KPIs, owners |
| **Single Source of Truth** | One agreed data model: entities, statuses, vocabulary, all departments aligned |
| **Escalation Matrix** | Four-level decision framework defining what requires Founder/Owner vs what is delegated |
| **30/60/90 Roadmap** | Post-audit action plan: Foundation → Pilot → Rollout with gate criteria |
| **MVL Gates** | Minimum Viable Launch — pre-launch checkpoints that must pass before GO |
| **Spoke-and-Hub** | Structural model for multi-division resource allocation |
| **30% Day-1 Rule** | ~30% of funding goal must be achievable on Day 1 for algorithmic momentum |
| **Stupidity Tax** | Built-in contingency buffer (5-20%) for unknowns and execution risk |
| **Valley of Death** | Mid-campaign slump requiring proactive content and community ops |
| **Golden Sample** | Final production sample (factory-identical) locked before mass production |
| **KPI Scorecard** | "Nervous System" dashboard — traffic-light health check on one screen |
| **Campaign Update Engine** | Systematic mid-campaign content protocol preventing Valley of Death |
